# 基本数学运算示例

## 1. 四则运算

### 加法
```
a + b = 和
示例：5 + 3 = 8
```

### 减法
```
a - b = 差
示例：10 - 4 = 6
```

### 乘法
```
a × b = 积
示例：7 × 6 = 42
```

### 除法
```
a ÷ b = 商
示例：15 ÷ 3 = 5
```
<!--more-->
## 2. 指数运算

### 乘方
```
aⁿ = a × a × ... × a (n次)
示例：2³ = 2 × 2 × 2 = 8
```

#### 特殊指数规则：
- `a⁰ = 1` (a ≠ 0)
- `a¹ = a`
- `a⁻ⁿ = 1/aⁿ`

## 3. 开方运算

### 平方根
```
√a = b 当且仅当 b² = a
示例：√9 = 3
```

### n次方根
```
ⁿ√a = b 当且仅当 bⁿ = a
示例：³√27 = 3
```

## 4. 运算优先级（PEMDAS法则）

运算按以下顺序进行：

1. **P** - 括号
2. **E** - 指数
3. **M** - 乘法
4. **D** - 除法
5. **A** - 加法
6. **S** - 减法

```
示例：3 + 4 × 2² ÷ (1 + 1)
      = 3 + 4 × 4 ÷ 2
      = 3 + 16 ÷ 2
      = 3 + 8
      = 11
```

## 5. 运算性质

### 交换律
- 加法：a + b = b + a
- 乘法：a × b = b × a

### 结合律
- 加法：(a + b) + c = a + (b + c)
- 乘法：(a × b) × c = a × (b × c)

### 分配律
- a × (b + c) = a × b + a × c

## 6. 特殊运算

### 阶乘
```
n! = 1 × 2 × 3 × ... × n
示例：4! = 1 × 2 × 3 × 4 = 24
```

### 取模运算
```
a mod b = a除以b的余数
示例：17 mod 5 = 2
```

| 符号 | 名称 | 示例 |

| + | 加号 | 3 + 2 = 5 |

## 8. 计算示例

### 示例1：综合运算
```
计算：2 + 3 × 4² ÷ 2 - 5
步骤：
1. 指数：4² = 16
2. 乘法：3 × 16 = 48
3. 除法：48 ÷ 2 = 24
4. 加法：2 + 24 = 26
5. 减法：26 - 5 = 21
结果：21
```

### 示例2：带括号运算
```
计算：(5 + 3)² ÷ 4 × √16
步骤：
1. 括号：5 + 3 = 8
2. 指数：8² = 64
3. 开方：√16 = 4
4. 除法：64 ÷ 4 = 16
5. 乘法：16 × 4 = 64
结果：64
```
